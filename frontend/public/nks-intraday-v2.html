Futures-only simulation for the tradable window (CSV date ≥ 20-Mar-2026, FUT contracts only — no EQ fallbacks). Each trade is exactly 1 lot. Entry at 10:15 IST priced at session VWAP (09:15 → 10:15). Stop is a 15-minute VWAP flip — we exit when a 15-min bar closes below the session VWAP as of that bar's end. This is a tape-based, volume-weighted stop that moves with the rally, cuts losers cleanly, and avoids 1-min wick whipsaws.

Hard floors — trade is SKIPPED if any fail:
· ORB pass: 10:15 candle opens ≥ 15-minute OR high (09:15 → 09:30).
· Confidence ≥ 50 / 100 from the composite score below.
No fixed stop-cost gate — VWAP flip is a dynamic stop, not a fixed-distance one.
Confidence score (0–100):
· ORB margin (25): how far 10:15 clears OR high.
· Tightness (25): (OR high − OR low) / entry, as %. Tight range = clean structure, typically smaller VWAP-flip losses.
· VWAP slope (20): 09:45 vs 10:15 session VWAP rising = momentum building.
· Volume buildup (20): volume in 10:00–10:15 vs early-session average.
· Not extended (10): entry within 1% above OR high (not chasing a blow-off).
Exit logic (realistic, no oracle):
· Walk 15-min grid boundaries after 10:15 (10:30, 10:45, …, 15:15).
· At each boundary, if the prior bar's close is below the session VWAP up to that close — flip, exit at that close.
· Otherwise hold; exit at 15:15 close.
· No "best slot" cherry-picking. No 1-min tick stop-outs.
